### `yarn start` with dependencies:

1. Redux + React-Redux
2. Bootstrap + React-Bootstrap
3. React-Router-Dom
4. React-Icons

Make a .env file with [REACT_APP_API_KEY=#_YOUR_KEY_FROM_TMDB]

Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
